ℹ️ About SLECS Explorer

SLECS Explorer is an interactive platform for constructing and exploring Skeleton-Level Chemical Space (SLECS) from tandem mass spectrometry (MS/MS) data.

The platform reorganizes MS/MS spectra according to fragment-feature patterns identified through nonnegative matrix factorization (NMF). This representation helps reveal structurally coherent regions, distinguish chemically divergent areas, and prioritize chemically distinct regions that may contain previously unrecognized molecular scaffolds.

The online version provides an interactive demonstration using representative datasets. The local version supports the complete analytical workflow, including data preprocessing, matrix construction and filtering, NMF decomposition, visualization, annotation mapping, and parameter adjustment.
